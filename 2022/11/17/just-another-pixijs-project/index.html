安装依赖

首先安装 gulp： npm install -g gulp

安装完成后，使用 npm install --save <package name> 安装依赖包，由于我们用 ts 构建项目，推荐安装以下包：

  • @types/node
  • vinyl-source-stream
  • vinyl-buffer
  • gulp-typescript
  • typescript
  • browserify
  • browser-sync
  • fast-glob
  • pixi.js
  • fs
